Debugging:

Open DevTools (F12) and check the Console tab for messages like:

  • [Season Color Checker] Initializing...
  • [Season Color Checker] ColorThief available: true
  • [Season Color Checker] Settings loaded: {selectedSeason: "spring", filterEnabled: true}
  • [Season Color Checker] Found X product images to analyze

If you don't see these messages, the extension isn't loading properly.
